Lists

Metro 4 provides special classes to create any styled lists.

About

Metro 4 contains any styled lists: step-list, items-list, feed-list, custom-marker.

Custom marker

You can create list with custom marker. To create it, add class .custom-list-marker to your list element and set attribute data-marker for list item.

  • Home
  • Base
  • Widgets
  • Contacts

                    <ul class="custom-list-marker">
                        <li data-marker="★">Home</li>
                        <li data-marker="☄">Base</li>
                        <li data-marker="✓">Widgets</li>
                        <li data-marker="❤">Contacts</li>
                    </ul>

#欢迎
在Windows地铁网络开发项目的前端框架，创建并支持Sergey Pimenov（又名Alton）与社区的巨大支持和参与。Metro UI CSS开发与微软的建议来构建用户界面，包括：总的风格，网格，布局，字体，20 + 300 +组件，内置的图标。用{ {少建地铁CSS界面}。Metro UI CSS是开源的，麻省理工许可模式。
